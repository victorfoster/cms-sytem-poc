# Destination Celebrant
Simple content driven website playground for server side JavaScript rendered site

## Environment Setup

* install vscode
	- install the following extensions: prettier, CSScomb, tslint
* from command line, run ` setx ASPNETCORE_ENVIRONMENT "Development"`


## Developing

Generally you can just run `npm run dev`. When you do this will set up proxy that 
points //localhost:5000 and will automatically reload the 
browser when you save.
